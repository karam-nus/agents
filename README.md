# 🤖 AI Agents — The Complete Guide

> **From Zero to Agent Developer**: Understanding, building, and deploying AI agents & agentic workflows. A comprehensive learning path for ML practitioners entering the agentic world.

## Who This Is For

You know machine learning. You understand transformers, LLMs, and prompt engineering. But **agents** — autonomous systems that reason, use tools, and take actions — are new territory. This guide takes you from "what is an agent?" all the way to deploying production agentic applications and landing a job building them.

## 📚 Prerequisites

Before diving in, you should be comfortable with:

- **Python** — classes, async/await, decorators, package management
- **LLM fundamentals** — transformers, attention, autoregressive generation, prompting
- **API basics** — REST APIs, JSON, authentication tokens
- **Command line** — terminal usage, environment variables, pip/conda
